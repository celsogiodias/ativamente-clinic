# AtivaMente+ - Gestão de Clínica

Plataforma web para gestão completa de clínicas e consultórios, desenvolvida com HTML, CSS, JavaScript e Firebase.

## 🎯 Funcionalidades

- **Autenticação**: Login e cadastro de usuários com Firebase Authentication
- **Gestão de Pacientes**: Cadastro, visualização e organização de pacientes
- **Dashboard**: Resumo com informações principais da clínica
- **Agenda**: Sistema de agendamento de consultas (em desenvolvimento)
- **Prontuário**: Registro de atendimentos e histórico (em desenvolvimento)
- **Financeiro**: Controle de receitas e despesas (em desenvolvimento)
- **Criptografia**: Proteção de dados sensíveis com criptografia XOR e SHA-256

## 📋 Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Conta Firebase com projeto configurado
- Conexão com internet
